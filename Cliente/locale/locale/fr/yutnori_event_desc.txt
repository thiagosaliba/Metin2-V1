BIENVENUE AU JEU DE YUTNORI METIN2 ![ENTER]
[ENTER]
Chassez des monstres pour collecter des troncs de bouleau. Avec 28 troncs de bouleau,[ENTER]
vous obtenez un plateau de jeu de Yutnori : que les jeux commencent ! Ce jeu vous réserve[ENTER]
de formidables prix ![ENTER]
[ENTER]
LES RÈGLES DU JEU[ENTER]
[ENTER]
1. Vous jouez contre l'ordinateur.[ENTER]
2. Chaque joueur dispose de deux pions. Le premier à les avancer jusqu'à l'arrivée a[ENTER]
gagné.[ENTER]
3. On lance des bâtons de Yutnori. Vous avancez du nombre de cases correspondant au[ENTER]
nombre de bâtons qui atterrissent sur le côté plat.[ENTER]
4. Le premier lancer permet de déterminer lequel des deux joueurs commence. Celui qui[ENTER]
obtient le score le plus faible commence.[ENTER]
5. Cliquez sur « Lancer ». Votre tirage s'affiche en bas. Choisissez le pion que vous[ENTER]
voulez faire avancer.[ENTER]
6. Si vous avancez jusqu'à une case à flèche, vous pouvez prendre ce raccourci.[ENTER]
[ENTER]
LANCERS ET TIRAGES[ENTER]
[ENTER]
Do, un côté plat vers le haut : avancez d'1 case.[ENTER]
Gae, deux côtés plats vers le haut : avancez de 2 cases.[ENTER]
Geol, trois côtés plats vers le haut : avancez de 3 cases.[ENTER]
Yut, quatre côtés plats vers le haut : avancez de 4 cases.[ENTER]
Mo, quatre côtés ronds vers le haut : avancez de 5 cases.[ENTER]
Duit-do, bâton marqué avec côté plat vers le haut : reculez de 1 case. Si vous n'avez pas[ENTER]
de pion sur le plateau de jeu, vous devez passer un tour.[ENTER]
[ENTER]
Job-gi : si vous avancez jusqu'à la case du plateau de jeu où se trouve déjà[ENTER]
l'ordinateur, vous sortez son pion et gagnez 10 points.[ENTER]
Up-gi : si vos deux pions finissent leur course sur la même case, vous les avancez tous[ENTER]
les deux ensemble à partir de votre prochain tour.[ENTER]
Lancer encore une fois : si vous lancez Yut ou Mo, vous rejouez lorsque votre tour se[ENTER]
termine.[ENTER]
[ENTER]
Vous pouvez conférer une plus grande probabilité à un lancer donné. Pour ce faire,[ENTER]
sélectionnez-le dans le menu correspondant.[ENTER]
[ENTER]
POINTS[ENTER]
[ENTER]
1. Lancer des bâtons de Yutnori : -10[ENTER]
2. Capture du pion de l'ordinateur : +10[ENTER]
3. Capture de votre pion : -10[ENTER]
4. Votre pion arrive à destination : +10[ENTER]
5. Le pion de l'ordinateur arrive à destination : -10[ENTER]
[ENTER]
REMARQUE[ENTER]
[ENTER]
Si vous arrêtez le jeu avant la fin ou si vous quittez la carte sur laquelle vous vous[ENTER]
trouvez, vous n'obtenez pas de récompense.
